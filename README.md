# Laboratorio 4: SGD Y NN

Este laboratorio implementa desde cero dos componentes fundamentales del Machine Learning.  

En el Task 2.1 comparamos tres variantes de descenso por gradiente (Batch, Stochastic y Mini-batch) para ajustar un polinomio de grado 3 a datos ruidosos, analizando el trade-off entre velocidad de convergencia y estabilidad.  

En el Task 2.2 se construye un perceptrón binario para clasificar especies de flores Iris, visualizando cómo aprende una frontera de decisión lineal.  

Todas las implementaciones usan únicamente NumPy, sin librerías de alto nivel como scikit-learn, PyTorch o TensorFlow.
